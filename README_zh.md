# Org-GTD-Complete: 为 Emacs org-mode 设计的完整 GTD 工作流系统

[English](README.md) | [中文](README_zh.md)

本包在 org-mode 中完整实现了《搞定》（GTD）方法论的全部精髓，不仅包含任务处理的五个核心步骤，还完整实现了六层视野（Horizon of Focus）、自然计划模式、每周回顾、情境分类、两分钟原则以及所有标准GTD清单（项目、下一步行动、等待、将来/可能、参考资料）。此外，还包括收件箱处理、项目规划、回顾系统、参考资料管理、触发清单、头脑风暴工具、组织原则以及垂直和水平关注管理。它帮助用户从日常操作到人生目标进行全面管理，真正遵循GTD的核心教义。

## 安装

```elisp
M-x package-install org-gtd-complete
```
